Project Overview: My Stylish Web Page

Built using: HTML5 & Embedded Styling Elements

This project was developed to practice layout hierarchy, custom decorative styles, and multi-prompt mathematics script rules.

Key Features Implemented:

  • Linear Color Banner: Outlined with a soft pink to green background header blend pattern.
  • Typography Control Rules: Structured with custom word-spacing values, underline decorations, and italic header flags.
  • Calculation Script Loops: Runs alert and input prompt sequences to gather dynamic numbers and sum calculations instantly.
